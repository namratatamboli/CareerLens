# CareerLens

CareerLens is a Chrome Extension that helps users identify potentially fraudulent job postings by analyzing job details and generating a fraud risk assessment.

> 🚧 **Status:** Ongoing Project — actively being enhanced with improved UI, smarter detection logic, and Machine Learning features.

## Features

- Analyze job postings from supported websites
- Generate a fraud risk score
- Display potential risk indicators
- Provide safety recommendations
- Flask-based backend with Chrome Extension frontend

## Tech Stack

- Python
- Flask
- JavaScript
- HTML
- CSS
- Chrome Extension (Manifest V3)

## Upcoming Features

- Machine Learning-based fraud detection
- Company verification
- Salary analysis
- Support for more job portals
- Detection history
