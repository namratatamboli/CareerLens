# CareerLens

CareerLens is a Chrome extension that analyzes online job postings and detects potentially fraudulent listings using a Flask backend and AI-assisted risk analysis.

## Features
- Extracts job details from job portals
- Generates fraud risk score
- Displays explainable risk indicators
- Flask REST API backend

## Tech Stack
- Python
- Flask
- JavaScript
- HTML
- CSS
- Chrome Manifest V3
